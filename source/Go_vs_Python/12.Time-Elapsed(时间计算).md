#  Time-Elapsed(时间计算)

## Python

```python
import time

t0 = time.time()
time.sleep(3.5)  # for example
t1 = time.time()
print("Took %.2f seconds" % (t1 - t0))
```



## Go

```go
package main

import (
	"fmt"
	"time"
)

func main() {
	t0 := time.Now()
	time.Sleep(time.Second)
	elapsed := time.Since(t0)
	fmt.Printf("Took %s", elapsed)		//Took 1.0005267s
}
```
