Args
====

To run this:

::

   go run args.go peter anders bengt

And it should output:

::

   PETER
   ANDERS
   BENGT

------

Go
--

.. code:: go

   package main

   import (
       "fmt"
       "os"
       "strings"
   )

   func transform(args []string) {
       for _, arg := range args {
           fmt.Println(strings.ToUpper(arg))
       }
   }

   func main() {
       args := os.Args[1:]
       transform(args)
   }
