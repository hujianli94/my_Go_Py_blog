# 网页多媒体

## 网页多媒体的基本语法
``` 
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>多媒体的基本语法</title>
</head>
<body>
下面是mp3文件
<br>
<! --多媒体标记-->
<embed src="smak.slow.mp3">
</body>
</html>
```
循环播放
``` 
<embed src="欢子、沈丹丹 - 怀念过去.mp3" loop="true">
```
隐藏面板
``` 
多媒体面板是隐藏的，打开网页自动播放。
<embed src="欢子、沈丹丹 - 怀念过去.mp3" hidden="true">
```
设置面板空间大小
``` 
<embed src="欢子、沈丹丹 - 怀念过去.mp3" width="100px" height="1000px" bgcolor="black">
```
