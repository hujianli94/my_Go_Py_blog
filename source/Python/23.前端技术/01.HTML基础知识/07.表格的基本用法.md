## 表格的基本用法

## 表格的基本语法
``` 
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>表格的基本用法</title>
</head>
<body>
<! --设置表格和边框-->
<table border="1">
    <! --设置表格中的表头-->
    <tr>
        <! --设置表格中的表项-->
        <th>表头一</th>
        <th>表头二</th>
    </tr>
    <tr>
        <td>表元二</td>
        <td>表元二</td>
    </tr>
</table>

</body>
</html>
```
