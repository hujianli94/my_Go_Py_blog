# 类的方法

### 定义方法、并进行实例化

    类中定义方法和普通函数一样 def 方法名称(self)
     self表示类本身，必须写，名字约定俗成，可以修改，但是不建议修改
     使用方法前必须先实例化类
     hu = My_class()
     hu.方法名

#### 代码示例
```
#!/usr/bin/env python
#-*- coding:utf8 -*-

class smplclass:
    def info(self):
        print("我定义的类")

    def mycacl(self,x,y):
        return x + y


hu = smplclass()
print("调用info的方法！")
hu.info()
print("".ljust(50,"-"))
print("调用mycalc方法")
print(hu.mycacl(2,4))
```
