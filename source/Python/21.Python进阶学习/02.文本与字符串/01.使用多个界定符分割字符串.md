# 使用多个界定符分割字符串

``` 

line = 'asds asdsad; dsadasd,dasd,dasdasd,hujijjj'

import re

print(re.split(r'[;,\s]\s*', line))
```

输出信息的
``` 
['asds', 'asdsad', 'dsadasd', 'dasd', 'dasdasd', 'hujijjj']
```


``` 
fields = re.split(r'(;|,|\s)\s*',line)
values = fields[::2]
delimiters = fields[1::2] + ['']
print(values)
print(delimiters)
```

输出信息
``` 
['asds', 'asdsad', 'dsadasd', 'dasd', 'dasdasd', 'hujijjj']
[' ', ';', ',', ',', ',', '']
```
